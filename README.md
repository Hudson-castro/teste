UBS Salvador System - Vite + React + TypeScript

Como usar:
1. Unzip e entre na pasta: cd ubs-salvador-system
2. Instale dependências: npm install
3. Rode em modo dev: npm run dev
4. Abra http://localhost:5173

Observações:
- O arquivo original que você me enviou foi colocado em src/pages/Home.tsx e preservado integralmente.
- Esse projeto inclui configuração básica do Tailwind (tailwind.config.cjs, postcss.config.cjs).
- Se o seu código usa assets externos (imagens), certifique-se que as URLs estejam acessíveis.
